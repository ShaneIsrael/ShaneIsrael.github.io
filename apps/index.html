Snipping Tool++

Snipping Tool++ is a program I wrote to to replace the default Snipping Tool program that ships with the Windows OS. The program acts just like the original Windows Snipping Tool, with the added feature to upload your snippets and/or screenshots to Imgur and retrieve a link for sharing. You tool will also save a copy of the upload locally incase you ever need one along with the sharing url and deletion url. Like the Windows Snipping Tool, Snipping Tool++ has an editor with the added bonus of being able to "blur out" sensitive information.

Created with Java. Runs on Windows 7. (Windows 8 support coming soon)
