Android SSL ClientJune 2014

Android SSL Client is an SSL Client that I developed for wolfSSL. SSL Client uses the wolfSSL JNI Library which is just a Java wrapper for their native CyaSSL library. The Android SSL Client provides uses with a simple and easy way to test secure connections to user specified servers. You have the options to use your own certificates. You can change the security layer to use varying versions of SSL, TLS, and DTLS. The client also gives you access to a wide variety of Cipher Suites that you can choose from.


The Android SSL Client is available for free on the Google Play store. If you have any questions or suggestions please send them to info@wolfssl.com

Created with Java. Runs on Android.

Snipping Tool++April 2012

Snipping Tool++ is a program I wrote to to replace the default Snipping Tool program that ships with the Windows OS. The program acts just like the original Windows Snipping Tool, with the added feature to upload your snippets and/or screenshots to Imgur and retrieve a link for sharing. The tool will also save a copy of the upload locally incase you ever need one along with the sharing url and deletion url.


Like the Windows Snipping Tool, Snipping Tool++ has an editor with the added bonus of being able to "blur out" sensitive information.

Created with Java. Runs on Windows
