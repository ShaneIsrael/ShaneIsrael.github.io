Pixel RainMarch. 2014

Pixel Rain is a medium to fast paced block breaking combo smashing puzzle game. The goal is to create massive combos by chaining together rain blocks of the same color. The bigger the combo the bigger your reward. The game ends if either of your columns of rain blocks breach the hazard zone. Will you risk breaching the hazard zone to increase your combo? Small increases in combo size reward heavily in this game bringing a new meaning to risk vs reward in puzzle games. The game is currently available on the Google Play and Amazon App stores.

Created with Java using the LibGDX Framework. Runs on Android

Twitch Plays Crystal VersionFeb. 2014

This concept was not an original idea of mine, I was bored one day and wondered if I could create a Twitch stream that allows viewers to control Pokemon Crystal Version by inputing commands on the Twitch chat. I created a bot that monitored and intercepted commands from the twitch chat stream, the bot then sends the commands to the emulator which has Crystal Version running. Finally I am using Open Broadcast Software to stream the emulator to said channel. To keep the game from being a complete anarchy of random commands only the most popular command will picked and sent to the emulator.
-Commands-

  • start
  • select
  • a "presses the 'a' button"
  • b "presses the 'b' button"
  • left "moves the player left once"
  • right "moves the player right once"
  • up "moves the player up once"
  • down "moves the player down once"

    • Twitch streams have a 15-40 second video lag meaning your command may not show up until that time.

    Created with Java

    StratofallNov. 2013

    Stratofall was a game I made for my independent study project on Android programming. You play as a skydiver whose goal is to fall as far as possible dodging lightning clouds and popping balloons along the way. You control the skydiver by tilting your phone either left or right. The more balloons you pop before you run out of lives the higher your score will be.

    Created with Java and libGDX. Runs on Android.

    Pixel Rain OriginalJune 2011

    Pixel Rain was a game I built the summer after my first year of college. I didn't have any form of internet access and out of boredom I thought it would be interesting to see what kind of game I could make without internet help. The end result was Pixel Rain. Pixel Rain is a game that uses strictly Java programming, there are no images being used in this game everything you see is Java programming using the graphics class. The sounds however were not programmed in.

    Created with native Java. Runs on Windows.
